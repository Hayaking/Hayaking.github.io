    # golang项目

    先复制go.mod/sum, 然后安装依赖，一般依赖不会怎么变化，让其在最底层，可以利用docker的缓存，只要依赖不变，每次打镜像的时候就不用重新下载依赖

    FROM golang:1.17.5-alpine3.15 as build

ENV GO111MODULE on
ENV GOPROXY https://goproxy.cn
