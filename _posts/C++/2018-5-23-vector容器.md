## 1.vector:支持快速随机访问

vector是C++标准模板库中的部分内容，中文偶尔译作“容器”，但并不准确。它是一个多功能的，能够操作多种数据结构和算法的模板类和函数库。vector之所以被认为是一个容器，是因为它能够像容器一样存放各种类型的对象，简单地说，vector是一个能够存放任意类型的动态数组，能够增加和压缩数据

##2.必须导入的头文件

```c++
#include <vector>
```

## 3.vector对象的定义

```c++
vector<int> v;
```

所有的容器都是类模板,因此在定义某种容器时,要在容器名后面加上一堆尖括号,里面写此容器存放元素的数据类型,

此处是int

## 4.容器的构造函数



| 构造函数示例 | 构造函数说明                                                 |
| ------------ | ------------------------------------------------------------ |
| C<T> c;      | 创建一个名为c的空容器.C是容器名,T是元素类型,c对象名.适用于所有容器 |
| C c(c2)      | 创建c2的副本c;c和c2具有相同的容器类型,并存放相同类型的元素,适用所有的容器 |
| C c(n,t)     | 用n个值为t的元素创建容器c,其中t必须为C的元素类型.只是用与序列式容器 |
| C c(n);      | 创建具有n个初始化值的元素.只适用于顺序容器                   |

```c++
vector<int> a;	//空
vector<int> b(a);	//拷贝a
vector<int> c(10,-1);	//10个元素,每个初始化为-1
vector<int> d(10);	//10个元素,每个初始化为0
```

## 5.vector对象的操作

| 操作             | 操作说明                            |
| ---------------- | ----------------------------------- |
| v.empty()        | 如果v为空, 则返回true,否则返回false |
| v.size()         | 返回v中元素的个数                   |
| v.push_back(t)   | 在v的末尾追加一个元素               |
| v[n]             | 返回v中第n个元素                    |
| v1=v2            | v1的元素替换为v2的元素              |
| v1==v2           | v1与v2相等,返回true                 |
| v1 = {a,b,c....} | 用列表中的元素,拷贝替换v1中的元素   |
| <,<=,>,>=        | 顾名思义,以字典序比较               |

## 6.向vector中添加元素

> c++标准要求vector应该能在运行时高效快速地添加元素.
>
> 因此既然vector对象能高效的增长,那么就没有必要在定义vector对象的时候设定其大小.
>
> 事实上,设定大小的话,性能可能更差,只有一种例外,就是当所有的元素值都一样.
>
> 一旦有元素的值不同,更有效的方法是定义一个空的vector对象,再在运行时向其中添加具体值

```c++
vector<int> a;	//空
for (int i = 0; i < 100; i++) {
	a.push_back(i);
}
```

> 需要注意的是,如果循环体内部包含有想vector对象添加元素的语句,则不能使用范围for循环(范围for循环内不应该改变所遍历序列的长度).
>
> > c++中的范围for循环 与 java中的foreach类似.

## 7.size_type

vector的size成员返回该对象元素的个数,返回值的类型是由vector定义的size_type类型.

> 要使用size_type,需要制定它是哪种类型的.vector的对象类型总是包含着元素的类型
>
> ```c++
> vector<int>::size_type	//正确
> vector::size_type	//错误
> ```

## 8.不能使用下标添加元素

```c++
vector<int> v;	//空
v[0]=1;	//错误
v.push_back(1);	//正确
```

由于v是一个空的vector对象,长度为0,不包含任何元素,因此就不能通过下标访问,赋值!
